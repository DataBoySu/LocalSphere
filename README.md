# LocalSphere
LocalSphere is a public information system designed to make local, district-level insights accessible to everyone.<br>
The platform combines shared public kiosks and companion mobile/web applications to serve users with and without personal internet access.<br>
It provides location-aware information such as public advisories, healthcare alerts, historical patterns, and essential local context through an intuitive map-based and conversational interface. <br>
The system is centrally managed, scalable, and designed for multilingual and low-barrier access across diverse regions.
